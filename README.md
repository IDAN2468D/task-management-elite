# 🚀 TaskMaster Elite: AI-Native Task Management Ecosystem

Welcome to the future of productivity! **TaskMaster Elite** is a premium, full-stack task management ecosystem designed for modern high-performers. Built with a mobile-first philosophy and a powerful Node.js backend, it integrates AI insights to help you manage your mental load and avoid burnout.

---

## 🏗️ Technical Architecture

### 📱 Frontend (Mobile)
- **Framework:** [Expo](https://expo.dev/) (React Native) + [Expo Router](https://docs.expo.dev/router/introduction/)
- **Styling:** [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- **Animations:** [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) & [Skia](https://shopify.github.io/react-native-skia/)
- **Visuals:** [Lucide Icons](https://lucide.dev/), Mesh Gradients, and Glassmorphic UI components.

### ⚙️ Backend (Server)
- **Runtime:** [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/)
- **Database:** [MongoDB Cluster](https://www.mongodb.com/) (ODM: Mongoose)
- **AI Engine:** Integrated with Google [Gemini Pro API](https://ai.google.dev/) for NLP task entry and predictive analytics.

---

## ✨ Key Features

- **Predictive Burnout Guard:** AI analysis of mental load and productivity trends.
- **NLP Quick-Add:** Natural language task drafting for frictionless planning.
- **Deep Work Focus Mode:** Immersive UI with layout transitions to boost concentration.
- **Multi-Project Kanban:** Seamless management of complex project boards.
- **Glassmorphic Aesthetic:** An ultra-premium "Deep Indigo" visual identity.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB connection string
- Gemini Pro API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/IDAN2468D/task-management-elite.git
   cd task-management-elite
   ```

2. **Frontend Setup:**
   ```bash
   cd frontend
   npm install
   npx expo start
   ```

3. **Backend Setup:**
   ```bash
   cd ../backend
   npm install
   # Create a .env file based on the environment rules
   npm start
   ```
